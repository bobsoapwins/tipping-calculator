# **App Name**: TipEase

## Core Features:

- Bill Input and Splitting: Input the bill amount and split it among the group. Also let users specify the tip percentage.
- Tip Calculation: Calculate the tip amount and the total amount per person.
- Result Display: Display the results clearly, showing the tip amount, total bill, amount per person, and individual contributions.
- Save/Load Calculations: Allow users to save and load previous calculations for recurring groups.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) for a calming and friendly feel.
- Secondary color: Dark blue (#00008B) for contrast and emphasis.
- Accent: Teal (#008080) for interactive elements and highlights.
- Clean and intuitive layout with clear sections for input, calculation, and results.
- Smooth transitions and animations when updating calculations and displaying results.
- Use recognizable icons for bill, tip, and people, enhancing the app's usability.

## Original User Request:
A tipping calculator app to help calculate tip on a bill amongst friends. Use a blue color scheme and include complex transitions and animations.
